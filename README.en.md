## 🧩 Integrations & API

Profferio exposes a clean REST API and an event-driven automation engine so it plugs into your stack. _The following are illustrative examples._

**Create a ticket via the API**

```http
POST /api/psm/tickets
Authorization: Bearer <token>
Content-Type: application/json

{
  "project": "SUPPORT",
  "summary": "New customer request",
  "priority": "high",
  "fields": { "category": "billing", "channel": "email" }
}
```

**Example automation rule** — auto-assign & comment when a high-priority ticket is created:

```json
{
  "trigger": "TICKET_CREATED",
  "conditions": [{ "field": "priority", "op": "equals", "value": "high" }],
  "actions": [
    { "type": "assign", "to": "team-lead" },
    { "type": "comment", "text": "Priority: pick up within SLA." }
  ]
}
```
